# CAPRUX — Ethnic Streetwear Brand

> "Doing the Impossible, with GOD's Help"

Landing page untuk brand streetwear CAPRUX — berbasis di Jawa Barat.

## Fitur
- Responsive design
- Product showcase dengan 3 koleksi
- Halaman detail produk (gallery, size guide, spesifikasi)
- Filosofi brand & documentary series
- Notifikasi email untuk produk "Coming Soon"

## Teknologi
- HTML5, CSS3, Vanilla JS
- Google Fonts
- Tawk.to Live Chat

## Struktur
- `index.html` — Halaman utama
- `product.css` & `product.js` — Komponen bersama
- 3 halaman detail produk
